Welcome

timeoutJS is a simple library to handle automatically 'timing out' a webpage after inactivity.
Initilizing a timer is as easy as passing a value in seconds for the timeout length, and a function to call when the timer runs out.

The timer is automatically refreshed and started from 0 whenever a user clicks anywhere on the webpage.


Documentation

timeout.newTimer(seconds, callback);

Initializes a timout counter that expires in the number of seconds provided. Callback is called once the timer has expired. 


timeout.newTimer(120, function() {
    console.log("Timer has expired!");
})
                        
timeout.stop();

Cancels a timeout counter. 


timeout.stop();
                        
timeout.refresh();

Manually reset the timeout counter back to 0. 


timeout.refresh();

Note: This is handled automatically whenever the user clicks anywhere on the page. Use this method to keep the session alive during other activities. 

timeout.pause();

Keeps the timer set, simply pauses the timeout event from being sent. 


timeout.pause();

Note: Useful during activities such as video playback when the user will not click for long periods of time. 

timeout.resume();

After pause(), calling resume() allows the timeout event to be sent again timeout event from being sent. 


timeout.pause();
                        
timeout.getTime();

Returns the interval that is set for a timeout. 


timeout.getTime();
